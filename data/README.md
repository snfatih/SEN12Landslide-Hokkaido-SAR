# Data

This directory contains references to the SEN12Landslide Hokkaido subset.
Raw data is not stored in this repository.

## Acknowledgements

This project makes use of the SEN12Landslide dataset:
https://github.com/PaulH97/Sen12Landslides

Parts of the code are adapted from publicly available implementations,
with appropriate attribution.

## Subset Details

- Region: Hokkaido, Japan
- Event: 2018 Eastern Iburi Earthquake
- Sensor: Sentinel-1 SAR
- Temporal depth: 4 acquisitions

## Download Instructions

Please refer to the official SEN12Landslide repository for download instructions.
